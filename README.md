# BetaTestSongs.com

A webapp built for EDM producers to anonymously share their tracks and get feedback. 

## Tech

### Client (SPA)
* React
* React Router
* Vite
* Tailwind CSS

### Server
* Nestjs
* Prisma
* Postgres
* Mixpanel (Analytics)

### Hosting & Deploys
* AWS (ECS/EC2/ECR/RDS/Cloudfront/S3/SES)
* Docker
* Terraform
